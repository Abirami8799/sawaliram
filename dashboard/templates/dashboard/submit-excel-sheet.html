Guidelines for filling out and submitting the Excel file

  • The Excel file template has help boxes that appear when you click or hover over a column. The help box provides an overview of the data that needs to be filled out in that column. Be sure to read it while filling out the Excel file.
  • Please check for spelling or grammatical errors before submitting the file.
  • Only fill out single values in the columns and avoid comma-separated lists or brackets in the fields.
  • If you need to enter a range of grades from which a question might have been asked, enter the average value instead of entering a value like ‘7-8’, instead enter ‘7.5’
  • If there is additional information or clarification that does not fit into any of the columns/fields, enter it into the ‘Notes’ field as a comma-separated list of entries.
  • Do not create multiple sheets in a single Excel file
  • You can open the drop-down menu in fields like ‘Context’ by clicking on the drop-down button next to the cell or by using the keyboard with ‘Alt + Down Arrow’.
  • You can copy the contents of the cell directly above the current cell by pressing [Ctrl + d] on Windows/Linux or [Command + d] on MacOS. Handy when entering a question with similar information as the previous entry.
